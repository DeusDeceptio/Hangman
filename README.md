# Виселица (Hangman)

Классическая игра "Виселица" — словесная головоломка, в которой игроку нужно угадать загаданное слово, предлагая буквы по одной. Проект реализован на Python с использованием графического интерфейса `tkinter`.

## Правила игры

1.  Система загадывает случайное слово (длину и сложность можно настроить).
2.  Игрок вводит букву. Если буква есть в слове, она открывается на всех соответствующих позициях.
3.  Если предложенной буквы нет в слове, к виселице добавляется часть тела (всего 6 ошибок: голова, туловище, 2 руки, 2 ноги).
4.  Игра продолжается до тех пор, пока игрок не отгадает всё слово или не будет «повешен» (6 ошибок).
5.  За каждую угаданную букву или слово целиком начисляются очки. Количество очков зависит от выбранной сложности и длины слова.
6.  В случае победы игрок может сохранить свой результат в таблицу лидеров, указав имя.
7.  В случае поражения показывается загаданное слово.

## Управление и интерфейс

- **Главное меню**: кнопки «Старт», «Настройки», «Таблица лидеров».
- **Игровое поле**:
    - Поле ввода буквы или слова.
    - Отображение виселицы и текущего состояния отгадывания.
    - Счётчик ошибок и текущий счёт.
    - Список ещё неиспользованных букв.
- **Настройки**: выбор сложности (от 1 — частые слова до 5 — редкие) и длины слова (от 5 до 15 букв).
- **Таблица лидеров**: отсортирована по убыванию очков и возрастанию времени.

## Запуск игры
```python
python main.py
```
## Технологии и библиотеки
Проект использует только встроенные библиотеки Python:

- `tkinter` — для графического интерфейса
- `requests` (Python 3.6 или выше) — для получения случайных слов с внешнего API 
- `json` — для сохранения настроек и таблицы рекордов
- `datetime` — для замера времени прохождения игры

## Структура файлов
- `Data.py` — работа со случайными словами, настройками и таблицей лидеров (чтение/запись JSON).
- `forms/base_form.py` — базовый класс для всех окон приложения.
- `forms/main_form.py` — главное меню.
- `forms/game_form.py` — основная игровая логика и отрисовка виселицы.
- `forms/settings_form.py` — окно настроек сложности и длины слова.
- `forms/leaderboard_form.py` — окно таблицы рекордов.
- `forms/congratulation_form.py` — окно поздравления с победой и ввод имени.
- `settings.json` — файл для сохранения настроек пользователя.
- `leaderboard.json` — файл для хранения таблицы рекордов.

## Примечания

- Слова берутся с публичного API `random-word-api.herokuapp.com`. При отсутствии интернета или ошибке API возвращается слово "error".
