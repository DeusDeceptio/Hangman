Класическая игра "Hangman" (также  "Виселица" или "Балда"). Словесная головоломка, где игроку нужно угадать слово, называя буквы по одной.
<p align="center">Правила:</p>
1. Система загадывает слово
2. Игрок предлагает букву, которая может входить слово.
3. Если такая буква есть в слове, то система пишет её над соответствующими этой букве чертами — столько раз, сколько она встречается в слове. Если такой буквы нет, то к виселице добавляется часть туловища (голова, туловище, 2 руки и 2 ноги).
4. Игрок продолжает отгадывать буквы до тех пор, пока не отгадает всё слово.
5. За каждый правильный ответ игрок получает очки в зависимости от выбранной сложности.
6. Если туловище в виселице нарисовано полностью, то игрок проигрывает, считается повешенным.
7. Если игроку удаётся угадать слово, он выигрывает и может добавить свои результы в таблицу лидеров.

<p align="center">Реализовано с использование втроенных библиотек:</p>
1. tkinter
2. requests
3. json
4. datatime
